# Architecture Protocol 3D Portfolio

This repository contains the "Architecture Protocol" 3D portfolio for **Daksha Dubey**, built with React, TypeScript, Three.js, React Three Fiber, and GSAP.

Live site: [https://dakshadubey.vercel.app/](https://dakshadubey.vercel.app/)

## Key Modernizations
- **Architecture Protocol Branded**: Professional terminal aesthetic and "Full-Stack Architect" positioning.
- **Dynamic GitHub Integration**: Live project fetching from [dakshdubey](https://github.com/dakshdubey) repository.
- **LinkedIn Activity Swiper**: Responsive carousel for professional updates.
- **3D Character Scene**: Custom purple "Protocol" lighting and stealth-black character styling.
- **Interactive Terminal**: Character-by-character typing animation with actual project documentation.

## Tech Stack
- **Frontend**: React 18, TypeScript, Vite
- **3D Engine**: Three.js, React Three Fiber, React Three Drei
- **Animation**: GSAP (GreenSock Animation Platform)
- **UI Components**: Swiper.js, React Icons

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/dakshdubey/dakshadubey-port.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development server:
   ```bash
   npm run dev
   ```

## Deployment
Built and optimized for deployment on Vercel.
```bash
npm run build
```
